# 3 en Ratlla PRO — macOS Liquid Glass UI

Aplicació web del joc **3 en ratlla (Tic-Tac-Toe)** desenvolupada amb **HTML, CSS i JavaScript pur**, amb una interfície inspirada en el llenguatge visual de **macOS (Liquid Glass / Glassmorphism)** i una **intel·ligència artificial basada en l’algoritme Minimax**.

Aquest projecte està concebut com a **projecte de frontend real**, orientat a portfolio, posant èmfasi en la qualitat del codi, el disseny d’interfície i l’experiència d’usuari.

---

## Característiques principals

- Interfície gràfica inspirada en macOS
  - Glassmorphism amb `backdrop-filter`
  - Transparències, profunditat i jerarquia visual
  - Disseny net i coherent

- Animacions i transicions
  - Interaccions suaus
  - Feedback visual en hover i victòria
  - Animacions no intrusives

- Intel·ligència artificial
  - Algoritme Minimax
  - Comportament òptim (la IA no pot ser derrotada)
  - Gestió correcta de torns i estat del joc

- Marcador persistent
  - Emmagatzematge local amb `localStorage`
  - La puntuació es conserva entre sessions

- Disseny responsive
  - Adaptació completa a mòbil, tauleta i escriptori
  - Compatibilitat amb navegadors moderns

- Feedback sonor
  - So en interaccions clau
  - Millora de l’experiència d’usuari

---

## Tecnologies utilitzades

- HTML5  
- CSS3 avançat  
  - Glassmorphism  
  - Animacions i transicions  
- JavaScript (Vanilla JS)  
  - Lògica del joc  
  - Algoritme Minimax  
  - Gestió de l’estat i persistència local  

No s’han utilitzat frameworks ni llibreries externes.
